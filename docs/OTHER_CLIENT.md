現在のwebsocket-clientに違う表示モードを付けてください。コマンドライン引数で指定できるものとします。
この表示モードはテキストだけを表示してください。

Partialは常に最後の行に逐次的に表示します。
Finalを受信したら該当のPartial部分は消去し、Finalのみを表示して改行します。

# 例

## 1.Partialが届く

（音声:こんにちわ、今何をしていますか）

```
[Now] こんにち いまなにしてすか
```

## 2.Finalが届き、次のPartialが届く

(音声:これから出かける所です)

```
[18:03:11] こんにちわ、今何をしていますか？
[Now] これからでかけるころです
```

## 3.次のFinalが届く

```
[18:03:11] こんにちわ、今何をしていますか？
[18:03:22] これから出かけるところです。
[Now]
```
